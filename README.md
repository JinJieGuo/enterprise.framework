#### 版本说明:
该版本为企业管理平台第三版(Java版)  
第一版:ASP.NET MVC EF 领域驱动  
第二版:前后端分离 ASP.NET WebApi Dapper + angular.js  
第三版:前后端分离 ASP.NET 微服务架构/java + angular 8
